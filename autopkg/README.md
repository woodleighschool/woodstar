# Woodstar AutoPkg Processors

AutoPkg processors that upload MunkiImporter output into Woodstar and, optionally, set where that software deploys.

Make them available to AutoPkg:

```sh
autopkg repo-add woodleighschool/woodstar
```

The processors are published under `com.github.woodleighschool.woodstar.processors`.

Setup, the recipe flow, targets, cleanup, and repo import are documented in the AutoPkg section of the Woodstar docs (`docs/content/autopkg/`).
